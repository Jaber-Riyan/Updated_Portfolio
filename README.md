# Jaber Ahmed Riyan - Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and blog posts.

## 🚀 Features

- **Home/Portfolio Page**: Introduction and featured projects
- **Projects Page**: Detailed view of all projects with filtering capabilities
- **Blogs Page**: Collection of technical blog posts
- **Project Detail Pages**: In-depth view of individual projects
- **Blog Detail Pages**: Full blog posts with syntax highlighting
- **Admin/Manage Page**: Interface for managing content (if applicable)
- **Responsive Design**: Works on all device sizes
- **Smooth Animations**: Powered by GSAP for engaging user experience
- **Redux State Management**: Centralized state for application data
- **TypeScript**: Type-safe codebase for improved reliability

## 🛠️ Technologies Used

- **Frontend Framework**: React 19 with TypeScript
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM v7
- **Styling**: Tailwind CSS with Tailwind Merge
- **Animations**: GSAP (GreenSock Animation Platform)
- **Build Tool**: Vite
- **Icons**: (If used, mention specific icon library)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd jaber-ahmed-riyan-portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## 🚀 Usage

To start the development server:
```bash
npm run dev
```

To build for production:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
